# Research: mui/base-ui

Source: https://github.com/mui/base-ui

## Observed signals

- Unstyled UI component library with accessibility-first behavior.
- Intended as a foundation for custom design systems.
- Mature package/docs/test monorepo ecosystem.

## Applied takeaways

- Keep behavior layer decoupled from visual opinions.
- Build wrappers that encode project-specific styling and variants.
- Use accessibility as a non-negotiable baseline.
