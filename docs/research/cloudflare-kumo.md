# Research: cloudflare/kumo

Source: https://github.com/cloudflare/kumo

## Observed signals

- README states Kumo is built on Base UI.
- Supports top-level imports and granular component imports.
- Includes docs-site workflow and component-centric development tooling.

## Applied takeaways

- Keep primitives and wrappers separable.
- Preserve ergonomic package exports.
- Treat accessibility and docs as delivery requirements, not extras.
