# Research: design-tokens/community-group

Source: https://github.com/design-tokens/community-group

## Observed signals

- Official DTCG specification repository.
- Emphasizes stable, focused, extensible token standards.
- Goal is cross-tool and cross-platform token interoperability.

## Applied takeaways

- Use DTCG-like token object fields (`$value`, `$type`, metadata).
- Preserve extensibility (`$extensions`) for tooling evolution.
- Keep token format portable and implementation-agnostic.
