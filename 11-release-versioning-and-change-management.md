# 11. 릴리즈/버저닝/변경관리 체계

## 목표
- 브레이킹 변경을 예측 가능하게 관리하고 안정적인 배포 주기를 유지한다.

## 실행 항목
1. SemVer 정책을 토큰/컴포넌트 각각에 정의한다.
2. changeset 기반 릴리즈 노트를 자동화한다.
3. 브레이킹 변경 시 마이그레이션 가이드를 필수화한다.

## 산출물
- versioning policy
- release checklist
- changelog 템플릿
